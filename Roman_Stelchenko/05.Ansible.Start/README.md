
## Homework Assignment 1: Setting Up Ansible
Ansible installation:
```
pip install ansible
```

Playbook: *ansible_project/playbooks/1_print_hello.yml*

Run command:
```
ansible-playbook playbooks/1_print_hello.yml
```

## Homework Assignment 3: Managing Users and Groups
Playbook: *ansible_project/playbooks/3_manage_users.yml*

Possible arguments: *user, user_group, user_password(must be hashed)*

Run command:
```
ansible-playbook -i inventory.yml playbooks/3_manage_users.yml -uroot -e user=roman -e user_group=ansible
```

## Alies set-up
```bash
# .bashrc

# User specific aliases and functions

alias rm='rm -i'
alias cp='cp -i'
alias mv='mv -i'
alias upod='kubectl get pod --all-namespaces'
alias unodes='kubectl get nodes'
alias uname='kubectl get namespaces'

///
source ~/.bashrc
```

# 07.Kubernetes

## Minikube commands

```bash
    7  minikube start
    8  kubectl cluster-info
    9  minikube plugins list
   10  minikube addons list
   11  minikube addons enable dashboard
   12  minikube addons enable ingress
   13  minikube addons list
   14  minikube dashboard --url
   15  kubectl proxy --address='0.0.0.0' --disable-filter=true  &
   16  minikube start
   17  minikube start --cpus=2 --memory='2g'
```

# Обновление конфигурации деплоймента

## Описание изменений

Этот PR содержит улучшения конфигурации деплоймента из предыдущей задачи:

### Добавленные возможности

1. **ConfigMap для index.php**
   - Создан ConfigMap с index.php, который отображает hostname pod в качестве заголовка первого уровня

2. **Key Pair в качестве Secrets**
   - Сгенерирована пара публичный/приватный ключ(test и test.pub)
   - Ключи добавлены в кластер с помощью SealedSecret
   - Использован механизм SealedSecret для безопасного хранения

3. **Валидация**
   - Проверено отображение index.php
   - Проверено отображение secrets

## Обновленные манифесты

В PR включены следующие обновленные манифесты:
- `apache-configmap.yaml` - конфигурация ConfigMap
- `sealedsecret.json` - зашифрованные secrets

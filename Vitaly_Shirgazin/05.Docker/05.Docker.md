# 05.Docker

## Dockerfile

```bash

FROM alpine:latest
RUN apk update \
    && apk upgrade \
    && apk add --no-cache python3 \
    && apk add --no-cache py3-pip \
    && pip install --upgrade pip \
    && pip install flask

COPY app.py /opt/

ENTRYPOINT FLASK_APP=/opt/app.py flask run --host=0.0.0.0 --port=8080

```

## Python app script
```python
import flask

app.config["DEBUG"] = True
```
